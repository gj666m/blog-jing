# 项目进度总结

> 最后更新：2026-04-29

---

## 项目目标

搭建综合型个人网站（博客 + 生活 + 书架 + 关于），Markdown 驱动内容，Astro 静态站点，部署在自有 VPS。

## 技术栈

Astro 5.x + React 19 + Tailwind CSS 4.x + Pagefind + Giscus + Umami + pnpm
